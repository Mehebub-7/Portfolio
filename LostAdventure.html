Lost Adventure

An immersive survival adventure game where players, as the sole survivor of a mid-air crash, navigate a mysterious land to rescue stranded passengers.

My Role

I led the development of the dynamic crafting system and implemented AI-driven wildlife interactions, enhancing both gameplay depth and realism.

Technologies

  • Singleton Pattern : Implemented Singleton Pattern to ensure centralized control and global access to key systems including UI, gameplay, level, and audio management.
  • Spritesheet : Collaborated with the UI artist to consolidate UI assets into a spritesheet for better optimization and rendering efficiency.
  • Texture Atlas: Collaborated with the game artist to consolidate texture assets into a texture atlas for better optimization and rendering efficiency.
  • Match3: Implemented a Match-3 game using NavMesh, where players must match candies to clear the path and progress forward.
  • Bridge: Developed a bridge-building game with NavMesh, where players construct bridges to create a path across obstacles.
  • SortAway: Created a directional cube-moving game where players must strategically move cubes to save fellow passengers.
  • Bridge Puzzle: Built a bridge rotation puzzle game using NavMesh, where players rotate bridge parts to form a complete crossing path.
  • Global Links: Designed a wire-connecting puzzle game where players link circuits to charge a phone before the battery drains, enabling them to call for help.
  • Block Colors: Implemented a block-sorting game where players group boxes by color to clear a path and advance.
  • Slide Puzzle: Developed a sliding rock puzzle game where players rearrange rocks to clear the way for rescuing trapped passengers.
  • Mirror Puzzle: Created a mirror-based puzzle game where players rotate mirrors to direct a laser beam that cuts down trees blocking the path, rescuing stranded passengers.
