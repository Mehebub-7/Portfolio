Poker Hands

An innovative poker training game that challenges players to predict the best possible hands using probability-based multipliers.

My Role

I led the development of the dynamic betting system, allowing players to place bets at various stages (pre-flop, post-flop, or after the turn), and implemented the Star Value currency system to automate bets and balance profits or losses across all hands.


Technologies

  • Singleton Pattern : Implemented Singleton Pattern to ensure centralized control and global access to key systems including UI, gameplay, level, and audio management.
  • Spritesheet : Collaborated with the UI artist to consolidate UI assets into a spritesheet for better optimization and rendering efficiency.
  • Texture Atlas: Collaborated with the game artist to consolidate texture assets into a texture atlas for better optimization and rendering efficiency.
  • Probability Algorithms: Developed probability algorithms to handle all variations of the poker game logic.
  • Firebase Firestore: Utilized Firebase Firestore to store and manage real-time leaderboard data.
