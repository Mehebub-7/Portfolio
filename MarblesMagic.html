Marbles Magic: Marbles Games

An engaging marble-shooting puzzle game where players aim to eliminate chains of colorful marbles by matching three or more of the same color.

My Role

I contributed to the development of the user interface (UI), UI animations and UI Effects.

Technologies

  • DOTween: Utilized DOTween for smooth and efficient UI animations.
  • 2D Sprite Animation: Implemented for character motion and actions.
  • Particle System: Used to enhance UI with dynamic visual effects.
